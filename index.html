Snake Game

Growing up as a kid, back when the Nokia 8210 was the phone everyone had, I always played this particular called Snake Game. The Snake Game is one of the default, if not the only default game, on those types of nokia phones. Back then it was my favorite and I played it quite a lot. Years past and I finally learn the basics of coding and for my CDT 208 class, Intro to Media Technology, I decided to recreate the game on Processing. Processing is an open source programming language meant for new media art and visual design. I recreated a fully functional version of the snake game from the ground up and added a few twist of mine own. Aside from the regular goal of eating the capturing an objected piece, once one is captured the speed of the snake increases thus making the game harder. I plan on adding different types of levels of the games, including obstacles. I also added music to it by creating a song in audacity. Repo
